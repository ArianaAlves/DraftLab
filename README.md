# DriftLab

Base inicial para um prototipo de jogo survival em Unity.

## O que foi preparado

- Estrutura inicial em `Assets/_Project/`
- Scripts base de movimento, coleta, crafting e UI
- Base de sistemas para personagem, roupas e missoes
- Base de menu inicial, lobby e visual de roupa por cor
- Guia rapido para montar a primeira cena
- `.gitignore` pronto para Unity

## Estrutura atual

- `Assets/_Project/Scripts/Gameplay`: logica de jogo do prototipo
- `Assets/_Project/Scripts/UI`: scripts simples de interface
- `Docs/GameVision.md`: direcao do jogo e pilares
- `Docs/PrototypeRoadmap.md`: roadmap do prototipo
- `Docs/UnitySetup.md`: passo a passo para montar e testar no Unity

## Importante

Este repositorio ainda nao contem os arquivos nativos de um projeto Unity criados pelo editor, como `ProjectSettings` e arquivos `.meta`.
Para finalizar a criacao do jogo:

1. Abra o Unity Hub
2. Crie o projeto `DriftLab` em 3D
3. Aponte o projeto para esta pasta ou copie `Assets/` e `Docs/`
4. Deixe o Unity gerar automaticamente os arquivos restantes
